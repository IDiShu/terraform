Этот код включает в себя объявления вывода (outputs) для отображения информации о созданных IAM-пользователях и EC2-инстансах в вашей Terraform-конфигурации :

1. Вывести все детали созданных IAM-пользователей в виде списка:
```
output "created_iam_users_all" {
  value = aws_iam_user.users
}
```
   - Этот вывод (`created_iam_users_all`) будет содержать список созданных IAM-пользователей, включая все их атрибуты.

2. Вывести только идентификаторы созданных IAM-пользователей в виде списка:
```
output "created_iam_users_ids" {
  value = aws_iam_user.users[*].id
}
```
   - Этот вывод (`created_iam_users_ids`) будет содержать список идентификаторов (ID) созданных IAM-пользователей.

3. Вывести пользовательский список, содержащий информацию о созданных IAM-пользователях в виде строки для каждого пользователя:
```
output "created_iam_users_custom" {
  value = [
    for user in aws_iam_user.users :
    "Username: ${user.name} has ARN: ${user.arn}"
  ]
}
```
   - Этот вывод (`created_iam_users_custom`) будет содержать список пользовательских строк, в которых указано имя пользователя и его Amazon Resource Name (ARN).

4. Вывести пользовательскую карту, сопоставляющую уникальные идентификаторы пользователя и их имена:
```
output "created_iam_users_map" {
  value = {
    for user in aws_iam_user.users :
    user.unique_id => user.id
  }
}
```
   - Этот вывод (`created_iam_users_map`) будет содержать карту, в которой ключами будут уникальные идентификаторы пользователей, а значениями - их идентификаторы (ID).

5. Вывести список имен пользователей с длиной имени равной 4 символам:
```
output "custom_if_length" {
  value = [
    for x in aws_iam_user.users :
    x.name
    if length(x.name) == 4
  ]
}
```
   - Этот вывод (`custom_if_length`) будет содержать список имен пользователей, у которых длина имени равна 4 символам.

6. Вывести карту инстансов EC2, сопоставляющую их идентификаторы и публичные IP-адреса:
```
output "server_all" {
  value = {
    for server in aws_instance.servers :
    server.id => server.public_ip
  }
}
```
   - Этот вывод (`server_all`) будет содержать карту, в которой ключами будут идентификаторы EC2-инстансов, а значениями - их публичные IP-адреса.

Эти объявления вывода помогают вам отобразить и получить информацию о созданных ресурсах и их атрибутах после выполнения Terraform-процесса развертывания. Вы можете использовать команду `terraform output` для получения этой информации после выполнения `terraform apply`.
